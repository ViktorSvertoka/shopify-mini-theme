# [ДЗ] Shopify: Theme Optimization

## Опис

### 1. Applied Lazy-Loading to All Non-Critical Images

- Усі зображення, що знаходяться нижче першого екрану, тепер використовують
  атрибут `loading="lazy"`.
- Зображення hero-блоку залишено без lazy-load для коректного показника LCP.
- Перевірено роботу через Lighthouse Diagnostics.
- Зменшено фізичні розміри зображень, що підвантажуються, щоб знизити загальне
  навантаження на мережу та прискорити рендер.

### 2. Converted Raw `<img>` HTML to Liquid Helpers

- Статичні HTML-зображення замінено на Liquid-хелпери:
  - `{{ image_url }}`
  - `{{ image_tag }}`

### 3. Ensured Full Responsiveness

Тему протестовано та налаштовано для:

- 320px (мобільні пристрої)
- 768px (планшети)
- 1280px (десктопи)

### 4. Improved Accessibility

- Додано `alt`-описи до всіх зображень, де їх бракувало.
- Підвищено контрастність текстових елементів, що не проходили аудит.
- Перевірено відповідність WCAG-критеріям через Lighthouse.

### 5. Improved Best Practices

- Використано рекомендовані Shopify-підходи:
  - defer/async для JS, де можливо
  - CDN-оптимізовані медіа
  - валідна розмітка та семантичні теги
- Видалено невикористовувані ресурси:
  - зайві JS/CSS
  - дублікати секцій
  - непотрібні локалі
  - іконки та демо-файли Dawn

### 6. Improved SEO

- Додано мета-тег `meta description` із коротким описом магазину.
- Перевірено коректність заголовків, alt-текстів, aria-атрибутів.
- Вирівняно структуру заголовків для покращення SEO-балансу.

### 7. Lighthouse Before/After Audit

Звіти Lighthouse наведено у README:

- підвищення Performance на 4-6 пунктів
- покращені Best Practices, Accessibility та SEO до значень 96–100

## Рекомендації по подальшій оптимізації

### Performance

- Встановити Swiper через пакетний менеджер (npm/pnpm) замість CDN або
  переписати його на нативний слайдер, оскільки використовується лише один
  слайдер.
- Прибрати Tailwind та переписати стилі на модульний CSS зі структурою та
  вкладеністю (CSS Nesting) для зменшення загальної ваги стилів.
- Оптимізувати логіку банер-слайдера:
  - зараз всі великі зображення підтягуються одночасно
  - можна залишити лише прев’ю першого слайду
  - інші завантажувати лениво при свайпі

### Accessibility

- Розширити aria-атрибути для інтерактивних елементів.
- Додати фокус-стили для навігації з клавіатури.

### Code Quality

- Відмовитись від Tailwind на користь:
  - модульних CSS-файлів
  - CSS nesting
  - мінімізації зайвого коду
- Стандартизувати структуру Liquid-шаблонів згідно з принципами Shopify Theme
  Style Guide.
